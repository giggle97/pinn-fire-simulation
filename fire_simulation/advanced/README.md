# PINN火灾预测模拟系统文档

## 1. 模拟环境

本程序模拟了一个包含**地形**和**风场**影响的火灾蔓延预测环境：

- **模拟区域**：10m × 10m 的矩形区域
- **地形特征**：中心 (5,5) 位置有一个高斯形状的小山丘
- **风场特征**：从左下方向右上方 (45度角) 吹的风
- **起火点**：左下角 (1,1) 位置

## 2. 数学公式

### 核心方程：Eikonal 方程

$$|\nabla t| = \frac{1}{R(x, y, \text{slope}, \text{wind})}$$

其中：
- $t$：火到达时间
- $\nabla t$：时间梯度（指向火的传播方向）
- $R$：火的传播速度（动态变化）

### 速度场计算

$$R = R_{base} \times (1 + K_{slope} \times \text{slope}) \times (1 + K_{wind} \times \cos\theta)$$

其中：
- $R_{base}$：基础燃烧速度（1.0 m/s）
- $K_{slope}$：坡度影响系数（3.0）
- $K_{wind}$：风影响系数（2.0）
- $\cos\theta$：传播方向与风向的夹角余弦

### 地形模型

使用高斯函数模拟山丘高度：
$$h(x, y) = \exp\left(-\frac{(x-5)^2 + (y-5)^2}{2\sigma^2}\right)$$  

坡度计算：
$$\text{slope} = \frac{\sqrt{(x-5)^2 + (y-5)^2}}{\sigma^2} \times h(x, y)$$  

其中 $\sigma = 2.0$（高斯函数标准差）

## 3. 模型

采用**物理信息神经网络 (Physics-Informed Neural Networks, PINN)** 模型：

- **原理**：将物理方程（Eikonal方程）作为损失函数的一部分，引导神经网络学习符合物理规律的解
- **优势**：无需大量标注数据，仅需少量边界条件即可训练
- **应用**：特别适合求解偏微分方程，如火灾蔓延、流体力学等问题

## 4. 神经网络

使用**前馈神经网络 (FNN)**：

- **网络结构**：2 → 60 → 60 → 60 → 60 → 1
  - 输入层：2个神经元（x, y坐标）
  - 隐藏层：4层，每层60个神经元
  - 输出层：1个神经元（火到达时间）
- **激活函数**：tanh
- **权重初始化**：Glorot uniform
- **优化器**：Adam（学习率0.001）
- **训练迭代**：20,000次
